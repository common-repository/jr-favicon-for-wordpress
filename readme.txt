== Description ==

JR Favicon for Wordpress is a very light-weight plugin that automatically adds your favicon (a small website icon) in the browser's title bar, making your blog easier to recognize. Also works in Internet Explorer, unlike some other plugins.


**Features:**

*   Makes your Wordpress blog easier to recognize
*   Works in every browser (including IE, Firefox, Chrome, Safari and Opera)
*   Very lightweight, small code
*   Favicon also shows up in browser bookmark list
*   It automatically configures itself! (The favicon.ico file is automatically selected from your website's root)


**Demo:**
Check out my website [PianoMusician.nl](http://www.pianomusician.nl/ "JR Favicon demo"), you can see the blue piano favicon.

== Installation ==

Installation is very easy:

1. Upload `jr-favicon.php` to the `/wp-content/plugins/` directory and activate the plugin
1. Create a favicon using [Dynamic Drive's favicon generator](http://www.pianomusician.nl/ "Generate a favicon"), and upload it to your site root (http://www.yoursite.com/favicon.ico).

That's it, your favicon will now automatically show up when visiting your blog! You can also enter an other favicon URL at the plugin page.
